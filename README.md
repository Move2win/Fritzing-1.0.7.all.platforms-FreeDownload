# How to Download

1. Ignore everything in the code section (those are outdated).

2. Go to the [_**Releases section**_](https://github.com/Move2win/Fritzing-1.0.7.all.platforms-FreeDownload/releases) to the right.

3. Download the latest version for your system.

4. Install it.

5. Enjoy!
